# Starship

Cross-shell prompt.

## Installation

```bash
brew install starship
```

## Configuration

Config: `~/.config/starship.toml` (symlinked via stow)

## Activation

Auto-loaded in `~/.zshrc`:

```bash
eval "$(starship init zsh)"
```

## Nerd Fonts

Install for icon support:

```bash
brew install --cask font-hack-nerd-font
```

Set as terminal font.

## Presets

View presets:

```bash
starship preset --help
```
