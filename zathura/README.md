# Zathura

Lightweight PDF viewer.

## Installation

```bash
brew tap zegervdv/zathura
brew install zathura zathura-pdf-mupdf zathura-pdf-poppler
```

## Configuration

Config: `~/.config/zathura/` (symlinked via stow)

## Usage

```bash
zathura document.pdf
```

Vim-style keybindings for navigation.
