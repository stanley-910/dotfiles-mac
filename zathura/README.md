# Zathura

Lightweight PDF viewer with vim-style keybindings.

## Installation

```bash
brew tap homebrew-zathura/zathura
brew install zathura zathura-pdf-mupdf
```

Note: Install only one PDF plugin (mupdf OR poppler), not both.

## Configuration

Config: `~/.config/zathura/` (symlinked via stow)

## Usage

```bash
zathura document.pdf
```

Vim-style keybindings for navigation.
